# アプリ名
SHUWA WE GO!!

# 概要
クイズ形式で楽しみながら手話を学べることを目的としたアプリです。  
手話の意味を選択して答えていくシンプルな仕組みです。

# 本番環境
https://shuwa-game.herokuapp.com/  
  

# 制作背景（意図）
楽しみながら手話を学んで欲しいという思いで作成しています。  
他の言語に比べて手話は学べるツールが少ないと感じたのがきっかけです。  
少しでも手話を学んでみたいと思っている方々が始めやすく、気がつけば習得できている！そんなサービスを目標にしています。

# 課題や今後実装したい機能
* コーチング機能<br>
（例）うんち記録のデータを元に食事のアドバイスや、近所の動物病院を紹介してくれるなどの機能です。

* 相談投稿のジャンル分け<br>
動物の種類、相談内容などで細かく分けて、よりユーザー同士でコミュニケーションをしやすくします。

* 相談投稿の検索機能<br>
ユーザーが相談投稿を検索できる機能です。

* 各ビューファイルの改修<br>
より、見やすく直感的に利用できるビューを作成します。
